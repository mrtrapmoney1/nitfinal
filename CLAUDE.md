## Project Overview

Static marketing website for **Nebraska Institute of Technology (NIT)** — an electronics training program operating in partnership with Metro TV and Appliance Service. Deployed at `https://nebnit.com`.

Built with **Next.js 15** (App Router, React Server Components, TypeScript). Run `npm run dev` for dev server, `npm run build` for production.

## Commands

```bash
npm run dev      # dev server at localhost:3000
npm run build    # production build + next-sitemap
npm run start    # serve production build
npm run lint     # ESLint
```

## Architecture

**App Router** with React Server Components. Pages are statically generated at build time with `export const revalidate = 86400` (ISR, 24h refresh).

**SEO**: each page exports `metadata` and `generateMetadata` with title, description, canonical, openGraph, and Twitter fields. Dynamic OG images via `/api/og` using `@vercel/og`.

**Email**: Resend client in `src/lib/resend.ts`. `/api/contact` and `/api/classes` route handlers validate, sanitize, and send email to `service@metrotv-audiotech.com`.

**GSAP**: client-only via dynamic `import('gsap')` inside `useEffect` in `GsapInit.tsx` — prevents SSR hydration errors.

**Active nav state**: `usePathname()` from `next/navigation` in `Header.tsx` — no `activePage` prop needed.

**JSON-LD schemas**: defined inline in each page file, passed to `SchemaOrg.tsx` server component which injects a `<script type="application/ld+json">` tag.

**CSS design tokens**: defined as custom properties on `:root` in `globals.css`. Use `--color-navy`, `--color-accent`, `--space-*`, `--radius-*`, `--shadow-*`, `--transition` etc.

## Environment Variables

Copy `.env.example` to `.env.local` and fill in values. Never commit `.env.local`.

```
RESEND_API_KEY              # required for contact/classes forms
NEXT_PUBLIC_GA4_MEASUREMENT_ID   # optional: Google Analytics 4
NEXT_PUBLIC_MAPS_EMBED_KEY       # optional: Maps Embed on contact page
GOOGLE_SERVICE_ACCOUNT_KEY       # optional: Search Console API (JSON string)
GOOGLE_PSI_API_KEY               # optional: PageSpeed Insights API
NEXT_PUBLIC_SITE_URL             # https://nebnit.com
```

## Deployment

Hosted on **Vercel** — push to `main` auto-deploys to `https://nebnit.com`. No manual deploy steps needed.

`vercel.json` contains 301 redirects from legacy `.html` URLs to preserve SEO.
